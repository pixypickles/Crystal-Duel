# Crystal Lane Battle AI

スマホ向けの1人プレイ・レーンシューティングゲームです。相手はAIです。

## 難易度

- イージー：移動・射撃・防御の反応が遅く、チャージ攻撃も少なめ
- ノーマル：標準的な反応速度と攻撃頻度
- ハード：従来版と同等の強敵AI

## 弾の性能

- A（低速）: 通常5 / チャージ8
- B（中速）: 通常4 / チャージ7
- C（高速）: 通常3 / チャージ6
- 各種類は画面内に1発ずつ

## 弾同士の衝突

弾の現在威力を差し引いて判定します。

- 同じ威力: 両方消滅
- 威力が異なる: 弱い弾は消滅し、強い弾は差分の威力で貫通
- 例: 威力8と威力3が衝突すると、威力5の弾が残る
- 貫通後の弾は残り威力に応じて小さくなります

## v5 ジャストガード
- ガード開始から0.16秒以内に被弾すると弾を反射します。
- 反射後も弾種・速度・残り威力を維持します。
- 反射弾は再度ジャストガードで打ち返せます。
- 0.16秒経過後は通常ガードになり、弾を消します。
