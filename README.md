# Crystal Lane Battle

5レーン制のローカル2人対戦シューティングゲームです。ブラウザだけで動作するため、GitHub Pagesへそのまま公開できます。

## 遊び方

| 操作 | Player 1 | Player 2 |
|---|---|---|
| 上下移動 | `W` / `S` | `↑` / `↓` |
| A（低速弾） | `A` | `J` |
| B（中速弾） | `D` | `K` |
| C（高速弾） | `F` | `L` |
| ガード | `G` | `I` |

- 攻撃キーを約0.7秒以上長押しして離すとチャージ弾になります。
- 通常弾はダメージ3、チャージ弾はダメージ5です。
- 同時に出せる弾は各プレイヤー3発までです。
- 同じ威力の弾同士は相殺します。チャージ弾と通常弾がぶつかった場合、チャージ弾は威力2・小サイズで貫通します。
- ガード中はキャラクターへの弾を防ぎます。
- ガードを310ms以内に2回押すと、現在レーンへ一撃だけ防ぐ壁を配置します。
- キャラクターは不死身ですが、被弾すると約0.45秒硬直します。
- クリスタルは中央3レーン分の当たり判定を持ち、HP20が0になると敗北です。

## GitHub Pagesへの公開

1. このフォルダの中身をGitHubリポジトリへアップロードします。
2. GitHubの `Settings` → `Pages` を開きます。
3. `Deploy from a branch` を選び、`main` / `(root)` を指定します。
4. 数分後に表示されるURLへアクセスします。
