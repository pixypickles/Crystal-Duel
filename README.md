# Crystal Lane Battle AI

スマホ横向き専用の1人プレイ・レーンシューティングゲームです。相手はAIです。

## 弾の性能

- A（低速）: 通常5 / チャージ8
- B（中速）: 通常4 / チャージ7
- C（高速）: 通常3 / チャージ6
- 各種類は画面内に1発ずつ

## 弾同士の衝突

弾の現在威力を差し引いて判定します。

- 同じ威力: 両方消滅
- 威力が異なる: 弱い弾は消滅し、強い弾は差分の威力で貫通
- 例: 威力8と威力3が衝突すると、威力5の弾が残る
- 貫通後の弾は残り威力に応じて小さくなります
